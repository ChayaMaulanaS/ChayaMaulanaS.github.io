👨‍💻 Tentang Saya

Saya merupakan lulusan Universitas Negeri Padang, Program Studi Pendidikan Teknik Informatika. Selama perkuliahan, saya menjalani magang di PT. Aka Solusi Teknologi dan melaksanakan Praktik Lapangan Kesekolahan di SMK Negeri 6 Padang. Saya memiliki ketertarikan yang besar di bidang IT, khususnya pada desain grafis. Saat magang, saya terlibat dalam pembuatan berbagai desain pamflet, termasuk dua di antaranya yang digunakan dalam event Job Fair di SMK Negeri 6 Padang.

Saya juga memiliki minat mendalam dalam pengembangan game, dan telah mengerjakan dua proyek game selama masa studi. Pada semester 5, saya mengembangkan sebuah game Visual Novel sebagai tugas akhir mata kuliah dan berhasil meraih nilai A. Selanjutnya, saya memilih Game Based Learning sebagai fokus tugas akhir perkuliahan dengan membuat game edukatif ber-genre Role Playing Game (RPG) untuk siswa kelas X di SMK Negeri 6 Padang, dan kembali meraih nilai A untuk proyek tersebut.

Di luar bidang akademik, saya memiliki bakat dalam bermain gitar sejak SMP dan saat duduk di kelas 2 SMA, saya dipercaya mewakili Kabupaten Tanah Datar dalam ajang Festival Lomba Seni Siswa Nasional (FLS2N) tahun 2018 pada bidang gitar.
